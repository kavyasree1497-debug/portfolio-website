# portfolio-website
A personal portfolio website developed to showcase my skills, projects, resume, certifications, and contact details in a professional and responsive manner. The website highlights my technical expertise, project experience, and career profile with a clean UI and smooth navigation. It is designed to help recruiters and companies easily understand my background and technical capabilities.
Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and responsive design
JavaScript – Interactive features and animations
Bootstrap / Tailwind CSS – Responsive UI components
Git & GitHub – Version control and project hosting
Vercel / Netlify – Deployment of the website
